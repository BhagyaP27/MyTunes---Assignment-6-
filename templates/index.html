MyTunes+ Personal Music Collection Database

This custom Flask web interface wraps a local SQLite database for browsing artists, albums, tracks, playlists, and genres. It is designed to support the project's main use cases: finding all tracks by an artist, calculating playlist totals, exploring many-to-many relationships, and refining one query into another.
